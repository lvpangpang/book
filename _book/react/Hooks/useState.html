1 使用

1.1 使用方式

  1. 支持传入值
    const [count, setCount] = useStaet([1, 2])
setCount([1, 2, 3])
  2. 支持传入函数，函数有一个参数是最新的 count 值，所以多次调用都会有效果而不是只生效一次
    const [count, setCount] = useStaet([1, 2])
setCount((count) => {
  const temp = [...count]
  temp.push(3)
  return temp
})

1.2 和 this.setState()比较

this.setState({
  count: [1, 2],
})
this.setState(
  (state, props) => {
    return {
      count: state.count + 1,
    }
  },
  () => {
    console.log(this.state.count) // 获取最新的值
  }
)
  1. React 使用 Object.is(value1, value2) 比较算法来比较 state 对于复杂类型改变，比如数组，不管 this.setState()还是 useState()都需要传入一个新引用地址的数组才能让页面重新渲染
const arr1 = [1, 2]
const arr2 = arr1
arr2.push(3)
Object.is(arr1, arr2) // true, 判断相等，所以不会更新
  1. 与 class 组件中的 setState 方法不同，useState 不会自动合并更新对象。
  2. setState 方法传入相同值也会触发更新，useState 不会触发更新

4 简单源码实现

import React from 'react'
import ReactDOM from 'react-dom'

const state = [] // 存放变量状态
let stateIndex = 0

// 强制刷新视图
function render() {
  ReactDOM.render(<App />, document.querySelector('#root'))
}

function useState(initialState) {
  state[stateIndex] = state[stateIndex] ? state[stateIndex] : initialState
  const setState = (newState) => {
    state[stateIndex] = newState
    stateIndex = 0
    render()
  }
  const value = state[stateIndex]
  stateIndex++
  return [value, setState]
}
