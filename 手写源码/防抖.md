# 防抖
一定时间内，有新事件触发会把旧的未执行的事件干掉，重新等待新事件执行

```javascript
function debounce(fn, time) {
  let timer = null
  return function () {
    let _this = this
    let arg = arguments
    if (timer) { // 与节流的区别在这
      clearTimeout(timer)
    }
    timer = setTimeout(function () {
      fn.apply(_this, arg) // 绑定this， 否则this指向window
    }, time || 300)
  }
}
```
